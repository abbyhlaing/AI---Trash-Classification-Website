# AI-Powered Waste Classification System

## Overview
This project is an AI-powered waste classification system designed to classify waste images into six categories using deep learning and computer vision techniques. The system aims to support efficient waste segregation and promote environmental sustainability through automated image classification.

The application allows users to upload waste images through a web interface and receive real-time classification predictions generated by a Convolutional Neural Network (CNN).

---

## Features

- Real-time waste image classification
- Deep learning image recognition using CNNs
- EfficientNet-B2 transfer learning model
- Web-based image upload interface
- Entropy-based uncertainty detection
- Classification into 6 waste categories:
  - Cardboard
  - Glass
  - Metal
  - Paper
  - Plastic
  - General Waste

## Dataset
The model was trained using the TrashNet dataset containing over 2,500 labelled waste images.

### Dataset Split
- 80% Training
- 20% Testing

### Data Augmentation
- Random Rotation
- Horizontal Flip
- Image Normalisation

---

## Model Performance

| Metric | Result |
|--------|--------|
| Test Accuracy | 87.35% |
| Weighted F1-Score | 0.87 |
| Metal Recall | 94% |
| Glass Recall | 91% |

The system also implements Shannon Entropy uncertainty detection to identify low-confidence predictions.

## Project Workflow

1. User uploads a waste image
2. Image is sent to the backend server
3. AI model processes the image
4. Prediction result is generated
5. Classification is displayed on the frontend
